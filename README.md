# SAURABH-PORTFOLIO
This project is a personal portfolio website developed using ASP.NET Web Forms to showcase my profile, technical skills, and contact information. It demonstrates my understanding of ASP.NET architecture, Master Pages, and front-end web design.

🚀 Features
Responsive navigation using ASP.NET Master Page
Dedicated pages: Home, About, Contact
Clean and simple UI using HTML, CSS, and JavaScript
Well-structured ASP.NET Web Forms project architecture
Easy to customize and ready for deployment

🛠️ Technologies Used
ASP.NET Web Forms (.NET Framework)
C#
HTML5
CSS3
JavaScript
Visual Studio

📂 Project Structure
File	Description
Default.aspx	Home page
About.aspx	About me and skills
Contact.aspx	Contact information
Site.Master	Common layout and navigation
Web.config	Application configuration

🎯 Purpose of the Project
This project was created to:
Showcase my ASP.NET development skills
Demonstrate web design fundamentals
Serve as a starter template for future ASP.NET projects
